Run yarn example from the repo root — it builds the SDK and starts the dev server automatically. Credentials and base URL are read from sdk/.env.e2e (falls back to process.env in CI). Opening via file:// still works but env values won't be injected. In production use https://gopaycdn.com/js-sdk/gopay-sdk.min.js.
